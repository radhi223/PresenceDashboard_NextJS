# PresenceDashboard_NextJS

Panduan singkat untuk menyiapkan proyek dashboard kehadiran berbasis Next.js.

## Prasyarat
- Node.js 18+ dan npm terpasang.
- Akses internet untuk mengunduh dependensi.

## Langkah Setup
1. **Install dependensi**
	```bash
	npm install
	```
2. **Jalankan server pengembangan**
	```bash
	npm run dev
	```
	Aplikasi akan tersedia di http://localhost:3000.

## Struktur Direktori Singkat
- `app/` – Halaman dan layout Next.js.
- `components/` – Komponen UI dan layout.
- `hooks/` – Hook kustom yang digunakan ulang.
- `lib/` – Utilitas umum.
- `public/` – Aset statis.
- `styles/` – Berkas CSS global tambahan.
